## 🚀 使用方法

### 1. 启动开发服务器
```bash
cd mp3-transformer
npm run dev
```

### 2. 访问应用
打开浏览器访问：http://localhost:5176

### 3. 使用批量下载功能
1. **上传文件**：拖拽或点击选择加密音频文件
2. **开始转换**：点击"开始转换"按钮
3. **打包下载**：转换完成后，点击"打包下载"按钮
4. **解压使用**：解压下载的ZIP文件
